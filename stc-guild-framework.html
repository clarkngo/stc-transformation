The STC Guild

A community of practice focused on mastery, mentorship, and continuity.

The Rank System

Solving the leadership gap through natural progression.

RANK 01

Apprentice

New members learning the ropes. They join existing project teams to gain hands-on skills (HOS) and contribute to documentation.

RANK 02

Journeyman

Experienced builders. They serve as Project Leads for 3-month sprints, managing GitHub repos and mentoring Apprentices.

RANK 03

Grandmaster

The Core Hub. They handle the "Guild Hall" administration, funding, and external relations, ensuring the path is clear for the builders.

The Guild Hall

Instead of fragmented silos, all communication happens in one central hub. When a project completes, it isn't deleted—it is Archived.

  • ✓ Centralized Discord: One home for Dev, Cyber, and AI.
  • ✓ The Library: A permanent wiki/GitHub org that stores every "Handover Doc" from previous generations.

The Succession Cycle

Weeks 1-2: Kickoff
Party up and select "Project Spokes."
Weeks 3-8: The Build
Intensive sprint aligned with HOS sessions.
Weeks 9-10: The Showcase
Demo the work and finalize the Archive.

The Continuity Clause

"No project shall begin without a designated Chronicler. If a Project Lead graduates, the Chronicler ensures the code, bugs, and vision are moved to the Library. We do not restart; we resume."

Adopted to ensure that no student’s hard work is ever lost to time.
